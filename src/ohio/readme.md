# Jordan Scrapes Secretary of States: Ohio

This package has some sample code getting data from the [Ohio Secretary of Search Business Search](https://businesssearch.ohiosos.gov/). 

A simple search and scrape of Ohio businesses using incrementing ids to get the most recently registered businesses.

## Getting Started

Clone the repository and run `npm i`. 

After that, you just need to run `npm start`.

[Full Guide Here](https://javascriptwebscrapingguy.com/jordan-scrapes-secretary-of-states-ohio/)

### Prerequisites

Tested on Node v12.4.0 and NPM v6.9.0.
