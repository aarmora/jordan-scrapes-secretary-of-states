# Jordan Scrapes Secretary of States: Vermont

This package has some sample code getting data from the [Vermont Secretary of Search Business Search](https://bizfilings.vermont.gov/online/BusinessInquire). 

A simple search and scrape of Vermont businesses using incrementing ids to get the most recently registered businesses.

## Getting Started

Clone the repository and run `npm i`. 

After that, you just need to run `npm start`.

[Full Guide Here](https://javascriptwebscrapingguy.com/jordan-scrapes-secretary-of-states-vermont/)

### Prerequisites

Tested on Node v12.4.0 and NPM v6.9.0.
